  1. กำหนดพลังงานไฟฟ้าที่ต้องการใช้
  2. ตั้งค่าเวลาแจ้งเตือน
  3. ตรวจจับไฟฟ้าที่ใช้
  4. แจ้งเตือนตามเวลาที่ตั้งพร้อมทั้งคำนวณไฟฟ้าที่ใช้ให้เป็นเงิน โดย อ้างอิงค่าไฟฟ้าจาก การไฟฟ้าส่วนภูมิภาค


Input - พลังงานไฟฟ้าที่ต้องการใช้ (Target Limit) - ตั้งเวลาการแจ้งเตือน (Time)


Process - ตรวจจับไฟฟ้าที่ใช้ และคำนวณออกมาเป็นเงิน หากใช้ไฟฟ้าเกิน จะได้ค่าเงินมาติดลบ หากใช้ไฟฟ้าน้อยกว่ากำหนดจะเงินมาเป็นค่าบวก


Output - ค่าไฟฟ้าที่ใช้เกินหรือไม่ และ เงินที่ประหยัดได้ในแต่ล่ะวัน


ผลกาวิเคราะห์ตนเองกับโจทย์


ไม่ยากและไม่ง่ายเกินไปสำหรับผมครับ


ผมชอบซูโดโค้ดมากกว่าครับเพราะว่าดูมีความง่ายมากกว่าครับ


ยากที่สุด คือ ขั้นตอนการหาข้อมูล และการเขียนโค้ดคำนวณค่าไฟฟ้าออกมาเป็นเงินครับ แก้โดยถามจาก gemini ว่าเว็บไซต์ไหนน่าเชื่อถือบ้าง และให้ gemini แก้โค้ดส่วนที่ผมพลาดและเพิ่มฟังก์ชั่นก์ที่สำคัญ
